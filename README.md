### Run Locally

To run **Portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/flaffycandie/Realportfolio.Brian
```

Windows:

```bash
git clone https://github.com/flaffycandie/Realportfolio.Brian
```
